## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-11 | Tampering | Delete from detail page | mitigate | Server action validates auth and checks entity exists before deleting; $transaction ensures atomicity |
| T-03-12 | Spoofing | Detail page route | mitigate | `getDesigner`/`getGenre` call `requireAuth()` as first operation |
| T-03-13 | Information Disclosure | Non-existent entity ID | mitigate | `notFound()` returns 404, does not expose whether entity "almost" exists |
| T-03-14 | Denial of Service | Malformed ID in URL | accept | Prisma safely handles non-existent IDs (returns null); `notFound()` called; no crash |
</threat_model>
