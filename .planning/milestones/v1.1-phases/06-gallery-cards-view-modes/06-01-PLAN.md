    Create `src/components/features/gallery/use-gallery-filters.ts` as a `"use client"` module.

    Use nuqs `useQueryState` with typed parsers per the RESEARCH.md Pattern 3. Check Context7 for the exact nuqs API if uncertain about import paths.

    ```typescript
    'use client';

    import { useQueryState, parseAsString, parseAsStringLiteral, parseAsArrayOf } from 'nuqs';
    import { useMemo, useDeferredValue } from 'react';
    import type { GalleryCardData, ViewMode, SortField, SortDir } from './gallery-types';
    import { VIEW_MODES, SORT_FIELDS, SORT_DIRS } from './gallery-types';
    import { compareFn } from './gallery-utils';
    ```

    Define parsers:
    - `view`: `parseAsStringLiteral(VIEW_MODES).withDefault('gallery')`
    - `sort`: `parseAsStringLiteral(SORT_FIELDS).withDefault('dateAdded')` (per D-13)
    - `dir`: `parseAsStringLiteral(SORT_DIRS).withDefault('desc')` (per D-13)
    - `search`: `parseAsString.withDefault('')`
    - `status`: `parseAsArrayOf(parseAsString, ',').withDefault([])`
    - `size`: `parseAsArrayOf(parseAsString, ',').withDefault([])`

    Export `useGalleryFilters(cards: GalleryCardData[])` hook returning:
    ```typescript
    {
      // URL state
      view: ViewMode;
      sort: SortField;
      dir: SortDir;
      search: string;
      statusFilter: string[];
      sizeFilter: string[];

      // Setters
      setView: (v: ViewMode) => void;
      setSort: (field: SortField) => void;  // toggles dir if same field, sets default dir if new field
      setDir: (d: SortDir) => void;
      setSearch: (s: string) => void;
      toggleStatus: (s: string) => void;    // adds or removes from array
      toggleSize: (s: string) => void;      // adds or removes from array
      clearFilters: () => void;             // resets search + status + size only

      // Computed
      filteredAndSorted: GalleryCardData[];
      totalCount: number;
      filteredCount: number;
      hasActiveFilters: boolean;
    }
    ```

    Implementation details:
    - `setSort(field)`: if field === current sort, flip dir. If new field, set sort=field and dir to 'desc' for dateAdded, 'asc' for all others.
    - `toggleStatus(s)`: if s in statusFilter, remove; else add. Use `setStatusFilter`.
    - `toggleSize(s)`: same toggle logic.
    - `clearFilters()`: call `setSearch('')`, `setStatusFilter([])`, `setSizeFilter([])`.
    - `filteredAndSorted`: useMemo over cards, applying:
      1. Search filter: `debouncedSearch` (use `useDeferredValue(search)`) on card.name, case-insensitive `.includes()`
      2. Status filter: if statusFilter.length > 0, filter by `statusFilter.includes(card.status)`
      3. Size filter: if sizeFilter.length > 0, filter by `sizeFilter.includes(card.sizeCategory)`
      4. Sort: `[...filtered].sort(compareFn(sort, dir))`
    - `hasActiveFilters`: `search !== '' || statusFilter.length > 0 || sizeFilter.length > 0`
