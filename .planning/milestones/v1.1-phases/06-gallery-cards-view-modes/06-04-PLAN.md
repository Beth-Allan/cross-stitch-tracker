    **Step 1: GalleryGrid (`src/components/features/gallery/gallery-grid.tsx`)**
    Mark as `"use client"`.

    Props:
    ```typescript
    interface GalleryGridProps {
      cards: GalleryCardData[];
      view: ViewMode;
    ```

    Render based on `view`:

    **Gallery view (`view === 'gallery'`):**
    ```tsx
    <div
      role="list"
      className="grid gap-6"
      style={{ gridTemplateColumns: 'repeat(auto-fill, minmax(280px, 340px))' }}
    >
      {cards.map(card => (
        <div key={card.chartId} role="listitem">
          <GalleryCard card={card} />
        </div>
      ))}
    </div>
    ```

    **List view (`view === 'list'`):**
    Compact rows. Each row is a link-wrapper div styled as a grid row:
    ```
    grid-template-columns: 8px minmax(180px, 2fr) minmax(120px, 1fr) minmax(100px, 120px) 64px 56px
    ```
    Columns per UI-SPEC:
    1. Status dot (colored circle, 8px)
    2. Name (font-heading text-sm font-semibold, truncate) + designer below (text-xs text-muted-foreground)
    3. Context stats: WIP = progress %, Unstarted = kitting summary, Finished = date
    4. Status label (text-xs)
    5. Size badge (text-[10px] uppercase)

    Each row: `bg-card border-b border-border py-2 px-4 hover:bg-muted/50 transition-colors`
    Row name is a Link to `/charts/${card.chartId}`.

    For small screens (< 640px), hide columns 4-5 and stack name/designer.

    **Table view (`view === 'table'`):**
    HTML `<table>` with sortable headers. Columns:
    - Project (name + designer) — always visible
    - Status — always visible
    - Designer — hidden max-md
    - Size — hidden max-sm
    - Progress — hidden max-sm (WIP: "45%", Unstarted: "--", Finished: "100%")
    - Stitches — hidden max-lg
    - Colours — hidden max-lg

    Table headers: `<th>` elements as `<button type="button">` for sortability per D-11:
    - Text: header label
    - Icon: if this column is current sort, show ChevronUp (asc) or ChevronDown (desc)
    - onClick: `onSortChange(fieldForColumn)`
    - aria-sort: "ascending" or "descending" when active, undefined otherwise
    - Styling: `text-muted-foreground text-xs font-semibold uppercase tracking-wider cursor-pointer hover:text-foreground transition-colors`

    Map column to SortField: Project -> 'name', Status -> 'status', Designer -> 'designer', Size -> 'size', Progress -> 'stitchCount', Stitches -> 'stitchCount', Colours -> not sortable (no sort)

    Table row styling: `hover:bg-muted/50 transition-colors border-b border-border/60`
    Name cell: Link to `/charts/${card.chartId}` with `text-sm font-semibold text-foreground hover:text-primary`
