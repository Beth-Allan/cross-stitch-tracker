# Supply Detail Modal

**What:** Add a read-only detail modal when clicking a supply in the catalog (thread, bead, or specialty item). Currently clicking goes straight to edit mode.

**From DesignOS:** `product-plan/sections/supply-tracking-and-shopping/supply-detail.png` and `SupplyDetailModal.tsx`

**Design shows:**
- Large color swatch with code + name
- "Used in X projects" list with project names as links
- Required/Acquired quantities per project
- Click-through to project detail

**Why post-MVP:** Edit-first is fine for building out the catalog. The detail modal becomes more valuable once there are real projects linked to supplies and you want to see "where is this thread used?" at a glance.
