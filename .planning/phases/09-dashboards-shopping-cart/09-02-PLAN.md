Create `src/lib/actions/project-dashboard-actions.ts` with `"use server"` directive.

**getProjectDashboardData()**: calls `requireAuth()`, then fetches all user projects via `prisma.project.findMany` with includes:
- `chart` (select: name, stitchCount, coverThumbnailUrl, designer.name, genres)
- `sessions` (select: date, stitchCount)
- `projectThreads` (select: id) — for count
- `projectBeads` (select: id) — for count
- `projectSpecialty` (select: id) — for count
- `fabric` (select: name, brand.name)

From the full project list, compute:

1. **heroStats**: Filter/aggregate from all projects:
   - `totalWIPs`: count where status === "IN_PROGRESS"
   - `averageProgress`: mean of `progressPercent` for IN_PROGRESS projects, 0 if none. progressPercent = `chart.stitchCount > 0 ? Math.min(100, Math.round((stitchesCompleted / chart.stitchCount) * 100)) : 0`
   - `closestToCompletion`: IN_PROGRESS project with highest progressPercent, null if none
   - `finishedThisYear`: count FINISHED/FFO with finishDate in `new Date().getFullYear()`
   - `finishedAllTime`: count FINISHED/FFO
   - `totalStitchesAllProjects`: sum all project.stitchesCompleted

2. **progressBuckets**: Assign non-finished projects to buckets:
   - `"unstarted"`: status is UNSTARTED, KITTING, or KITTED
   - `"0-25"`: 1-25% progress (status IN_PROGRESS or ON_HOLD)
   - `"25-50"`: 26-50%
   - `"50-75"`: 51-75%
   - `"75-100"`: 76-100%
   
   Each bucket: `{ id, label, range, count: projects.length, projects: BucketProject[] }`
   
   Labels: "Unstarted", "Just Getting Started", "Making Progress", "Over Halfway", "Almost There"
   Ranges: "0%", "1-25%", "25-50%", "50-75%", "75-100%"
   
   For each BucketProject: compute `lastSessionDate` from sessions[0]?.date, `stitchingDays` from `new Set(sessions.map(s => s.date.toISOString().split("T")[0])).size`

3. **finishedProjects**: Filter FINISHED/FFO, map to FinishedProjectData[]:
   - `startToFinishDays`: if both startDate and finishDate, `Math.ceil((finishDate - startDate) / (1000*60*60*24))`, else null
   - `stitchingDays`: unique session dates count
   - `avgDailyStitches`: `stitchingDays > 0 ? Math.round(totalStitches / stitchingDays) : 0`
   - `threadCount`: projectThreads.length
   - `beadCount`: projectBeads.length
   - `specialtyCount`: projectSpecialty.length
   - `fabricDescription`: `fabric ? fabric.brand.name + " " + fabric.name : null`
   - `genres`: chart.genres.map(g => g.name)
   - Default sort by finishDate DESC (per D-15)

Return `{ heroStats, progressBuckets, finishedProjects }` as `ProjectDashboardData`.
