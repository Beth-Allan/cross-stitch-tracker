Create `src/lib/actions/shopping-cart-actions.ts` with `"use server"` directive.

**getShoppingCartData()**: Calls `requireAuth()`. Queries `prisma.project.findMany` with:
- `where: { userId: user.id, status: { notIn: ["FINISHED", "FFO"] } }`
- Include chart (name, stitchesWide, stitchesHigh, coverThumbnailUrl, designer.name)
- Include projectThreads with thread (include brand)
- Include projectBeads with bead (include brand)
- Include projectSpecialty with specialtyItem (include brand)
- Include fabric (select name)

Map results to `ShoppingCartData`:
- `projects`: Map each project to `ShoppingCartProject` with supply counts from junction array lengths and `fabricNeeded = !project.fabric && chart.stitchesWide > 0 && chart.stitchesHigh > 0`
- `threads`: Flatten all projectThreads across projects into `ShoppingSupplyNeed[]` with `junctionId: pt.id, supplyId: pt.threadId, brandName: pt.thread.brand.name, code: pt.thread.colorCode, colorName: pt.thread.colorName, hexColor: pt.thread.hexColor, quantityRequired: pt.quantityRequired, quantityAcquired: pt.quantityAcquired, unit: "skeins", projectId: project.id, projectName: chart.name`
- `beads`: Same pattern with bead junction, `code: pb.bead.productCode, unit: "packs"`
- `specialty`: Same pattern with specialty junction, `code: ps.specialtyItem.productCode, colorName: ps.specialtyItem.colorName, unit: "packs"`
- `fabrics`: Projects where `fabricNeeded=true`, mapped to `ShoppingFabricNeed`

**updateSupplyAcquired(type, junctionId, acquiredQuantity)**: Calls `requireAuth()`.

1. Validate input with Zod: `z.object({ type: z.enum(["thread", "bead", "specialty"]), junctionId: z.string().min(1), acquiredQuantity: z.number().int().min(0) })`
2. Look up junction record with project include to check ownership:
   - thread: `prisma.projectThread.findUnique({ where: { id: junctionId }, include: { project: { select: { userId: true } } } })`
   - bead: `prisma.projectBead.findUnique(...)` same pattern
   - specialty: `prisma.projectSpecialty.findUnique(...)` same pattern
3. If not found: return `{ success: false as const, error: "Record not found" }`
4. **IDOR check**: If `record.project.userId !== user.id`: return `{ success: false as const, error: "Unauthorized" }`
5. Update: `prisma.projectThread.update({ where: { id: junctionId }, data: { quantityAcquired: acquiredQuantity } })`
6. `revalidatePath("/shopping")`
7. Return `{ success: true as const }`

Wrap in try/catch. On ZodError: return `{ success: false as const, error: zodError.errors[0].message }`. On other errors: console.error and return generic error.
