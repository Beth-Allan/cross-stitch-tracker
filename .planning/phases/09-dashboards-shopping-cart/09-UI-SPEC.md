## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, badge inner padding |
| sm | 8px | Compact element spacing, chip gaps, tab gap |
| md | 16px | Default element spacing, card padding, row gaps |
| lg | 24px | Section padding, page horizontal padding |
| xl | 32px | Page top padding, layout column gaps, hero stats margin-bottom |
| 2xl | 48px | Major section vertical gaps (between Currently Stitching and Start Next) |
| 3xl | 64px | Page-level section breaks (between full-width dashboard sections) |

Exceptions:
- Currently Stitching cards: fixed 280px width, 160px cover height (per DesignOS D-04)
- Hero stat cards: `minmax(140px, 1fr)` auto-fit grid (per D-14)
- Collection Stats sidebar: 260px fixed width on desktop (per DesignOS layout)
- Shopping-for bar: sticky `top: 0` with `z-index: 20` and 8px vertical / 16px horizontal padding
- Progress bar segments: `min-width: 24px` when count > 0 to ensure visibility

## Typography

| Role | Size | Weight | Line Height | Font |
|------|------|--------|-------------|------|
| Body | 14px (text-sm) | 400 (regular) | 1.5 | Source Sans 3 |
| Label / Micro | 11px (text-[11px]) | 700 (bold) | 1.4 | Source Sans 3, uppercase tracking-wider |
| Heading | 20px (text-xl) | 700 (bold) | 1.2 | Fraunces |
| Page Title | 28px (text-2xl) | 700 (bold) | 1.2 | Fraunces |
| Stat Number | 20px (text-xl) | 700 (bold) | 1.2 | JetBrains Mono, tabular-nums |

Additional rules:
- JetBrains Mono is reserved for numeric stat displays, quantity controls, and tabular data ONLY. Never for body text.
- Section headings use Fraunces with a 40px (w-10) emerald underline accent bar (h-0.5, rounded-full).
- Uppercase micro-labels (11px) used for stat card labels and category headers -- always pair with `tracking-wider`. The bold weight plus uppercase + tracking-wider provides sufficient differentiation from body text without needing a third weight.

| Role | Token | Usage |
|------|-------|-------|
| Dominant (60%) | `bg-background` (stone-50 light / stone-950 dark) | Page background, surfaces |

Accent reserved for:

- Section heading underline bars (emerald-400, 40px wide)
- Active tab indicator (border-emerald-500)
- Progress bar fill (emerald-400/500)
- Focus ring (ring token)
- "Shopping for" project chips (emerald-100 bg, emerald-800 text)
- Hero stat card backgrounds (emerald-50/60 bg, emerald-100 border)
- Checkmark icons for fulfilled supply items

Secondary accent (amber-500/600):
- "Rediscover This One" sparkle icon and label
- Spotlight section label (amber-600 text, uppercase tracking)

Status colors (7, from CSS custom properties):
- `--status-unstarted` (stone-500), `--status-kitting` (amber-500), `--status-kitted` (emerald-600)
- `--status-in-progress` (sky-500), `--status-on-hold` (orange-500)

- Unstarted: stone-300, 0-25%: amber-400, 25-50%: emerald-400

## Interaction Contracts

### Dashboard Tab Switching
- **Pattern:** nuqs `parseAsStringLiteral` with `?tab=progress` URL state
- **Default:** "library" tab (Main Dashboard)
- **Animation:** None (instant swap per existing Pattern Dive precedent)

### Currently Stitching Scroll
- **Scroll:** Horizontal with `scroll-snap-type: x mandatory`, `scroll-snap-align: start` on cards
- **Arrow buttons:** 36x36px circular, white bg, stone-200 border, shadow-md. Appear conditionally.
- **Scroll amount:** 300px per click, smooth behavior
- **Scrollbar:** Hidden (`scrollbar-width: none`, webkit override)

### Quick Add Menu
- **Trigger:** Emerald-tinted button with Plus icon + "Quick Add" + ChevronDown
- **Open state:** Button fills to emerald-600 bg, Plus rotates 45deg, ChevronDown rotates 180deg
- **Dropdown:** 200px min-width, white bg, rounded-xl, shadow-xl, appears below trigger
- **Backdrop:** Fixed inset overlay to capture outside clicks
- **Items:** 8 items. First item ("Log Stitches") has bottom border separator and emerald icon color. Rest have stone-400 icons.
- **"Log Stitches" action:** Opens existing LogSessionModal from Phase 8
- **Other actions:** Navigate to respective add forms (chart add, supply add, etc.)

### Spotlight Shuffle
- **"Check It Out" button:** Emerald-600 bg, white text, rounded-xl, navigates to project detail
- **"Shuffle" button:** Outline style, stone-200 border, RefreshCw icon. Calls server action for new random project.

### Collection Stats Sidebar
- **Desktop (md+):** Always expanded, vertical stat rows
- **Mobile (max-md):** Collapsible with ChevronDown toggle. Collapsed shows heading only. Expanded shows horizontal chip layout.

### Progress Bucket Accordions
- **Default state:** All collapsed (per D-13)
- **Toggle:** Click header to expand/collapse. Chevron rotates: collapsed = -90deg, expanded = 0deg.
- **"Show more":** Paginated at 10 items per bucket. "Show more (N remaining)" link at bottom.

### Finished Project Cards
- **Compact header:** Click to expand/collapse. ChevronDown rotates on toggle.
- **Expanded panel:** Stats grid (`auto-fill, minmax(140px, 1fr)`), genre tags, "View project details" link.
- **Single expanded at a time:** Expanding one collapses the previous.

### Shopping Cart Project Selection
- **Checkbox toggle:** Square/CheckSquare icons (lucide). Selected = emerald-600, unselected = stone-300.
- **Selected state:** Card border changes to emerald-300, bg to emerald-50/30.
- **"Select all" link:** Right-aligned above project list.
- **"Details" expand:** Per-project supply breakdown nested below the row.

### Shopping-for Bar
- **Empty:** ShoppingBag icon + "No projects selected" text
- **With selections:** "Shopping for:" label + emerald pill chips with X remove button + "Clear all" link
- **Sticky behavior:** Sticks to top of viewport with backdrop-blur and semi-transparent bg

### Quantity Control (Supply Tabs)
- **Stepper:** Minus / display / Plus buttons in a row
- **Display:** "acquired/total" in JetBrains Mono, clickable for direct input via prompt
- **Fulfilled state:** Green background (emerald-100), checkmark icon appears
- **Disabled states:** Minus disabled at 0, Plus disabled when acquired >= total

### Shopping List Checkoff
- **Row click:** Toggles checked state for entire row
- **Checked appearance:** Emerald border, emerald-50 bg, text gets line-through + muted color
- **Checkbox:** Square/CheckSquare icons
- **"Clear checked" button:** Resets all checked items

## Accessibility

- All interactive elements keyboard-accessible with visible focus rings (ring token)
- Accordion sections: use `aria-expanded` on trigger buttons
- Checkbox toggles: use `aria-checked` or semantic checkbox role
- Tab bars: standard tab pattern with `role="tablist"`, `role="tab"`, `aria-selected`
- Progress bars: `role="progressbar"`, `aria-valuenow`, `aria-valuemin`, `aria-valuemax`
- Stacked bar chart: include `title` attributes on segments for tooltip info
- Color swatches: include alt text or aria-label with color name
- Status is never conveyed by color alone -- all badges include text labels with dot indicators
- ScrollableRow arrow buttons: `aria-label="Scroll left"` on the left arrow, `aria-label="Scroll right"` on the right arrow
- Spotlight shuffle button: `aria-label="Shuffle spotlight project"` (supplements the visible "Shuffle Spotlight" label for screen readers)
