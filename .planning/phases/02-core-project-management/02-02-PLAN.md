    **Create src/lib/actions/upload-actions.ts** with `"use server"` directive -- per D-05, D-06, D-07, D-08, D-09, D-10:

    Three-step presigned URL pattern:

    1. **`getPresignedUploadUrl(input: unknown)`** -- Step 1: generate presigned PUT URL:
       - Auth check
       - Parse with `uploadRequestSchema.parse(input)`
       - Validate contentType: if `category === "covers"`, must be in `ALLOWED_IMAGE_TYPES`; if `category === "files"`, must be in `ALLOWED_FILE_TYPES`
       - Generate file key: `${category}/${projectId}/${nanoid()}-${fileName}` (per D-10)
       - Create PutObjectCommand: `new PutObjectCommand({ Bucket: R2_BUCKET_NAME, Key: key, ContentType: validated.contentType })`
       - Generate signed URL: `await getSignedUrl(getR2Client(), command, { expiresIn: 600 })` (10 min expiry)
       - Return `{ url, key }`
       - Wrap in try/catch -- if R2 not configured, return `{ success: false, error: "File storage is not configured. Cover photo and file uploads are unavailable." }`

    2. **`confirmUpload(input: { chartId: string; field: "coverImageUrl" | "coverThumbnailUrl" | "digitalWorkingCopyUrl"; key: string })`** -- Step 3: save key to database after client uploads:
       - Auth check
       - Validate field is one of the allowed fields
       - `prisma.chart.update({ where: { id: input.chartId }, data: { [input.field]: key } })`
       - `revalidatePath(\`/charts/${input.chartId}\`)`
       - Return `{ success: true }`

    3. **`getPresignedDownloadUrl(key: string)`** -- generate presigned GET URL for viewing/downloading:
       - Auth check
       - Create GetObjectCommand: `new GetObjectCommand({ Bucket: R2_BUCKET_NAME, Key: key })`
       - `await getSignedUrl(getR2Client(), command, { expiresIn: 3600 })` (1 hour)
       - Return `{ url }`

    4. **`deleteFile(key: string)`** -- remove file from R2:
       - Auth check
       - Create DeleteObjectCommand: `new DeleteObjectCommand({ Bucket: R2_BUCKET_NAME, Key: key })`
       - `await getR2Client().send(command)`
       - Return `{ success: true }`

    5. **`generateThumbnail(chartId: string, coverKey: string)`** -- per D-09, server-side thumbnail with sharp:
       - Auth check
       - Fetch the original image from R2 using GetObjectCommand
       - Process with sharp: `sharp(buffer).resize(400, 400, { fit: "cover", withoutEnlargement: true }).webp({ quality: 80 }).toBuffer()`
       - Upload thumbnail to R2 with key: `covers/${chartId}/thumb-${nanoid()}.webp`
       - Update chart record: `prisma.chart.update({ where: { id: chartId }, data: { coverThumbnailUrl: thumbnailKey } })`
       - Return `{ success: true, thumbnailKey }`

    Imports needed:
