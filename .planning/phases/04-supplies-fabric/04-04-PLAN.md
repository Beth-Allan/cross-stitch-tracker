## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client component -> Server actions | Form data from modal forms passed to server actions |
| localStorage -> Client state | View mode preferences read from localStorage (untrusted but low-risk) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-12 | Tampering | localStorage view mode | accept | Only affects visual preference (grid vs table), no security impact; validated to "grid" or "table" strings |
| T-04-13 | Tampering | hexColor in style attribute | mitigate | Hex validated by Zod regex before storage; rendered via `style={{ backgroundColor }}` which React auto-sanitizes |
| T-04-14 | Spoofing | Supply form submissions | mitigate | All form submissions go through server actions with `requireAuth()` |
</threat_model>
